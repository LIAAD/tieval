.datasets
=========

The datasets module makes it easy to download and import several corpus build for temporal information extraction.

Currently, the supported datasets are the following:
    - AQUAINT
    - EventTime
    - GraphEve
    - MATRES
    - MeanTime_English
    - MeanTime_Spanish
    - MeanTime_Dutch
    - MeanTime_Italian
    - Platinum
    - TCR
    - TDDiscourse
    - TempEval_2_Chinese
    - TempEval_2_English
    - TempEval_2_French
    - TempEval_2_Italian
    - TempEval_2_Korean
    - TempEval_2_Spanish
    - TempEval_3
    - TimeBank_1.2
    - TimeBank_Dense
    - TimeBankPT
    - TimeBank
    - TrainT
